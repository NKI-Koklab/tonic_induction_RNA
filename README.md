# Bulk RNA-seq data analysis in TONIC-1 and TONIC-2 trials
This repository contains code to reproduce findings based on the bulk RNA-seq data analysis from the manuscript 'Immunomodulatory capacity of chemotherapy to enhance sensitivity to anti-PD1 in metastatic TNBC: final results of the TONIC-1 and TONIC-2 trial' (under review in Nature Cancer).
